## REMOVED Requirements

### Requirement: Direct Markdown image editing in Visual Edit
**Reason**: Visual Edit no longer hosts direct text controls for image alt, destination, or title. Images are presented with a read-only caption; editing any image field requires switching to source editing.
**Migration**: To edit an image's alt text, destination, or title, switch to source editing mode (e.g. via the view-mode toggle or by clicking the image island to jump the cursor to its source).
